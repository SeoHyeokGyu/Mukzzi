# 캐릭터 시스템 상세

## 먹찌 생성

- 회원가입 직후 먹찌 이름과 초기 외형을 선택
- 모든 유저는 **Lv.1 기본 먹찌**로 시작

## 레벨 체계

| 레벨 구간 | 레벨당 필요 경험치 | 비고 |
|-----------|-------------------|------|
| Lv.1 ~ 10 | 100 XP | 초반 빠른 성장 체감 |
| Lv.11 ~ 20 | 200 XP | 중반 적응기 |
| Lv.21 ~ 30 | 350 XP | 후반 도전 구간 |
| Lv.31 ~ 50 | 500 XP | 장기 유저 목표 |

## 경험치 획득 규칙

| 행동 | 획득 XP |
|------|---------|
| 식사 1회 기록 | 30 XP |
| 하루 3끼 완료 | 50 XP (보너스) |
| 새로운 메뉴 첫 기록 | 20 XP (보너스) |
| 영양 밸런스 달성 (하루 기준) | 40 XP (보너스) |

## 변신 시스템

먹찌는 **누적 섭취 영양소 비율**에 따라 외형이 변신합니다.

| 변신 형태 | 조건 | 외형 컨셉 |
|-----------|------|-----------|
| 기본 먹찌 | 초기 상태 | 둥글고 순한 기본형 |
| 근육 먹찌 | 단백질 비율 40% 이상 | 단단하고 힘센 체형 |
| 채소 먹찌 | 식이섬유·비타민 비율 35% 이상 | 싱그럽고 건강한 외형 |
| 에너지 먹찌 | 탄수화물 비율 50% 이상 | 활발하고 통통한 외형 |
| 밸런스 먹찌 | 3대 영양소 편차 10% 이내 | 균형 잡힌 황금 외형 |

- 영양소 비율은 **최근 7일 기록 기준**으로 산정
- 조건을 벗어나면 이전 형태로 복귀 (도감에는 기록 유지)
- 변신 시 연출 애니메이션 재생 + 도감 자동 등록

## 패널티 시스템

| 미기록 일수 | 상태 | 영향 |
|------------|------|------|
| 1일 | 정상 | 변화 없음 |
| 2일 | 허기짐 | 먹찌 표정 변화 (슬픈 표정) |
| 3일 | 배고픔 | 푸시 알림 발송 + 홈 화면 배고픔 연출 |
| 5일 이상 | 기력 저하 | 경험치 획득량 50% 감소 (기록 재개 시 즉시 복구) |

- 레벨·변신 형태는 **하락하지 않음** (이탈 방지)
- 패널티는 식사 1회 기록 시 즉시 해제
