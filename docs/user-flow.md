# 핵심 사용자 플로우

```
앱 실행
  │
  ├─ 신규 유저 ──→ 회원가입/로그인 ──→ 먹찌 생성 (이름·초기 외형 선택)
  │
  └─ 기존 유저 ──→ 홈 화면 (먹찌 상태 + 오늘 식사 현황)
                      │
                      ├─ [메뉴 결정]
                      │     ├─ 랜덤 룰렛
                      │     ├─ 이상형 월드컵
                      │     └─ 상황별 필터 (날씨·기분)
                      │           │
                      │           ▼
                      │     메뉴 확정
                      │
                      ├─ [식사 기록]
                      │     ├─ 메뉴명 검색 입력
                      │     └─ 영양소 자동 매핑 (DB 연동)
                      │           │
                      │           ▼
                      │     기록 저장
                      │           │
                      │           ▼
                      │     ┌─────────────────────────┐
                      │     │  경험치 획득 연출        │
                      │     │  경험치 바 실시간 반영    │
                      │     │  (조건 충족 시) 레벨업!   │
                      │     │  (조건 충족 시) 먹찌 변신!│
                      │     └─────────────────────────┘
                      │
                      ├─ [기록 확인]
                      │     ├─ 먹부림 캘린더 (날짜별 식사 이력)
                      │     ├─ 먹찌 도감 (변신 컬렉션)
                      │     └─ 영양 밸런스 피드백
                      │
                      └─ [미기록 시]
                            │
                            ▼
                      3일 미기록 → 먹찌 배고픔 알림
                      상태 변화 (배고픔 외형 노출)
```
