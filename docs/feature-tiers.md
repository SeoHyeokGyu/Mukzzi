# 기능 단계별 개발 계획

## Tier 1 — MVP (출시 목표 기능)

> 룰렛·이상형 월드컵·상황별 필터로 메뉴를 고르고, 텍스트로 식사를 기록하면 먹찌가 성장합니다.

| 분류 | 기능 | 상세 |
|------|------|------|
| 메뉴 결정 | 랜덤 룰렛 | 먹찌가 대신 룰렛을 돌려주는 인터랙티브 액션 |
| 메뉴 결정 | 이상형 월드컵 | 두 메뉴 비교 토너먼트로 최종 메뉴 선택 |
| 메뉴 결정 | 상황별 필터 | 날씨·기분 태그 기반 맞춤 추천 |
| 식사 인증 | 텍스트 직접 입력 | 메뉴명 검색으로 간편 등록 |
| 식사 인증 | 음식 영양소 DB 연동 | USDA + 식약처 DB 연동, 즐겨찾기 관리 |
| 캐릭터 성장 | 먹찌 기본 변신 | 영양소 태그에 따른 캐릭터 변신 연출 (4~5종) |
| 캐릭터 성장 | 경험치 + 레벨업 연출 | 식사 등록 직후 경험치 바 실시간 반영 |
| 캐릭터 성장 | 패널티 시스템 | 3일 미기록 시 먹찌 배고픔 알림 및 상태 노출 |
| 기록 | 먹부림 캘린더 | 날짜별 먹찌 아이콘 + 식사 기록 |
| 기록 | 먹찌 도감 | 변신했던 먹찌 외형 컬렉션 |
| 피드백 | 영양 밸런스 피드백 | 텍스트 기반 다음 메뉴 추천 멘트 |
| 인증 | 회원가입 / 로그인 | JWT 토큰 발급 및 갱신 |
| 인증 | 소셜 로그인 | Kakao / Google / Apple OAuth |

## Tier 2 — 고도화 (MVP 이후 순차 개발)

> AI가 영양 현황을 분석해 메뉴를 추천하고, 사진·영수증으로 식사를 인증합니다. 친구와 공유하고 퀘스트·스트릭으로 리텐션을 강화합니다.

| 분류 | 기능 | 상세 |
|------|------|------|
| AI | 음식 사진 자동 인식 | Google Vision API로 음식명 + 영양소 자동 추출 |
| AI | 영수증 OCR | Naver Clova OCR로 메뉴명 자동 추출 |
| AI | AI 메뉴 추천 | 오늘 영양 현황 기반 GPT 추천 + 부족 영양소 룰렛 가중치 |
| AI | 먹찌 일기 | GPT 자동 한 줄 코멘트 |
| 리텐션 | 스트릭 시스템 | 연속 기록 일수 표시 |
| 리텐션 | 주간 퀘스트 + 보상 | 업적 달성 조건 실시간 체크 및 보상 지급 |
| 리텐션 | 식사 패턴 리포트 | 주간/월간 식사 시간 통계 분석 |
| 리텐션 | 스마트 넛지 알림 | 시간·날씨·전날 식단 분석 기반 FCM 푸시 알림 |
| 리텐션 | 시즌 이벤트 | 계절 챌린지, 한정판 캐릭터 외형 |
| 소셜 | 친구 타임라인 피드 | 친구 식사 기록 공유 및 리액션 |
| 소셜 | 먹찌 카드 SNS 공유 | 내 먹찌 카드 이미지 생성 후 공유 |
| 소셜 | 친구 초대 알림 | *"OO가 음식을 기록했어요!"* |
| 캐릭터 | 외형 커스터마이징 | 획득 아이템으로 먹찌 꾸미기 |
| 캐릭터 | 뱃지 & 칭호 | *"근육질 완성"*, *"전설의 캐릭터"* 등 |
| 캐릭터 | 레벨업 보상 | 새로운 의상 / 미니게임 / 필터 해금 |
| 편의 | 지도/배달앱 딥링크 | 네이버 지도·배민 URL 스킴 연동 |
| 편의 | 가격대 필터 | 예산 조건 기반 메뉴 추천 |
| 편의 | 홈화면 위젯 | 먹찌 배고픔 게이지 실시간 표시 |

## Tier 3 — 미래 확장 (장기 로드맵)

> 그룹 배틀·소셜 챌린지·위치 기반 퀘스트 등 확장 기능을 추가합니다.

- 대화형 AI 영양 상담 (프리미엄)
- 그룹 챌린지 생성 및 현황 집계
- 그룹 메뉴 합의 (카카오톡 공유 + 실시간 투표)
- 1:1 식단 배틀
- 위치 기반 로컬 퀘스트
- 냉장고 파먹기
- 바코드 스캔
- 음식 MBTI
- 식비 예산 관리
- 할인 쿠폰 가챠
- Apple Watch / Wear OS 연동
- 유저 메뉴 플레이리스트
