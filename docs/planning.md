# 기획

## 사용자 플로우

```
앱 실행
  │
  ├─ 신규 유저 ──→ 회원가입/로그인 ──→ 먹찌 생성 (이름·초기 외형 선택)
  │
  └─ 기존 유저 ──→ 홈 화면 (먹찌 상태 + 오늘 식사 현황)
                      │
                      ├─ [메뉴 결정]
                      │     ├─ 랜덤 룰렛
                      │     ├─ 이상형 월드컵
                      │     └─ 상황별 필터 (날씨·기분)
                      │           │
                      │           ▼
                      │     메뉴 확정
                      │
                      ├─ [식사 기록]
                      │     ├─ 메뉴명 검색 입력
                      │     └─ 영양소 자동 매핑 (DB 연동)
                      │           │
                      │           ▼
                      │     기록 저장
                      │           │
                      │           ▼
                      │     ┌─────────────────────────┐
                      │     │  경험치 획득 연출        │
                      │     │  경험치 바 실시간 반영    │
                      │     │  (조건 충족 시) 레벨업!   │
                      │     │  (조건 충족 시) 먹찌 변신!│
                      │     └─────────────────────────┘
                      │
                      ├─ [기록 확인]
                      │     ├─ 먹부림 캘린더 (날짜별 식사 이력)
                      │     ├─ 먹찌 도감 (변신 컬렉션)
                      │     └─ 영양 밸런스 피드백
                      │
                      └─ [미기록 시]
                            │
                            ▼
                      3일 미기록 → 먹찌 배고픔 알림
                      상태 변화 (배고픔 외형 노출)
```

## 캐릭터 시스템

### 먹찌 생성

- 회원가입 직후 먹찌 이름과 초기 외형을 선택
- 모든 유저는 Lv.1 기본 먹찌로 시작

### 레벨 체계

| 레벨 구간 | 레벨당 필요 경험치 | 비고 |
|-----------|-------------------|------|
| Lv.1 ~ 10 | 100 XP | 초반 빠른 성장 체감 |
| Lv.11 ~ 20 | 200 XP | 중반 적응기 |
| Lv.21 ~ 30 | 350 XP | 후반 도전 구간 |
| Lv.31 ~ 50 | 500 XP | 장기 유저 목표 |

### 경험치 획득 규칙

| 행동 | 획득 XP |
|------|---------|
| 식사 1회 기록 | 30 XP |
| 하루 3끼 완료 | 50 XP (보너스) |
| 새로운 메뉴 첫 기록 | 20 XP (보너스) |
| 영양 밸런스 달성 (하루 기준) | 40 XP (보너스) |

### 변신 시스템

먹찌는 최근 7일 섭취 영양소 비율에 따라 외형이 변신합니다.

| 변신 형태 | 조건 | 외형 컨셉 |
|-----------|------|-----------|
| 기본 먹찌 | 초기 상태 | 둥글고 순한 기본형 |
| 근육 먹찌 | 단백질 비율 40% 이상 | 단단하고 힘센 체형 |
| 채소 먹찌 | 식이섬유·비타민 비율 35% 이상 | 싱그럽고 건강한 외형 |
| 에너지 먹찌 | 탄수화물 비율 50% 이상 | 활발하고 통통한 외형 |
| 밸런스 먹찌 | 3대 영양소 편차 10% 이내 | 균형 잡힌 황금 외형 |

- 조건을 벗어나면 이전 형태로 복귀 (도감에는 기록 유지)
- 변신 시 연출 애니메이션 재생 + 도감 자동 등록

### 패널티 시스템

| 미기록 일수 | 상태 | 영향 |
|------------|------|------|
| 1일 | 정상 | 변화 없음 |
| 2일 | 허기짐 | 먹찌 표정 변화 (슬픈 표정) |
| 3일 | 배고픔 | 푸시 알림 발송 + 홈 화면 배고픔 연출 |
| 5일 이상 | 기력 저하 | 경험치 획득량 50% 감소 (기록 재개 시 즉시 복구) |

- 레벨·변신 형태는 하락하지 않음 (이탈 방지)
- 패널티는 식사 1회 기록 시 즉시 해제

## 기능 단계별 개발 계획

### Tier 1 — MVP (출시 목표)

> 룰렛·이상형 월드컵·상황별 필터로 메뉴를 고르고, 텍스트로 식사를 기록하면 먹찌가 성장합니다.

| 분류 | 기능 | 상세 |
|------|------|------|
| 메뉴 결정 | 랜덤 룰렛 | 먹찌가 대신 룰렛을 돌려주는 인터랙티브 액션 |
| 메뉴 결정 | 이상형 월드컵 | 두 메뉴 비교 토너먼트로 최종 메뉴 선택 |
| 메뉴 결정 | 상황별 필터 | 날씨·기분 태그 기반 맞춤 추천 |
| 식사 인증 | 텍스트 직접 입력 | 메뉴명 검색으로 간편 등록 |
| 식사 인증 | 음식 영양소 DB 연동 | USDA + 식약처 DB 연동, 즐겨찾기 관리 |
| 캐릭터 성장 | 먹찌 기본 변신 | 영양소 태그에 따른 캐릭터 변신 연출 (5종) |
| 캐릭터 성장 | 경험치 + 레벨업 연출 | 식사 등록 직후 경험치 바 실시간 반영 |
| 캐릭터 성장 | 패널티 시스템 | 3일 미기록 시 먹찌 배고픔 알림 및 상태 노출 |
| 기록 | 먹부림 캘린더 | 날짜별 먹찌 아이콘 + 식사 기록 |
| 기록 | 먹찌 도감 | 변신했던 먹찌 외형 컬렉션 |
| 피드백 | 영양 밸런스 피드백 | 텍스트 기반 다음 메뉴 추천 멘트 |
| 인증 | 회원가입 / 로그인 | JWT 토큰 발급 및 갱신 |
| 인증 | 소셜 로그인 | Kakao / Google / Apple OAuth |

### Tier 2 — 고도화 (MVP 이후 순차 개발)

> AI가 영양 현황을 분석해 메뉴를 추천하고, 사진·영수증으로 식사를 인증합니다.

| 분류 | 기능 | 상세 |
|------|------|------|
| AI | 음식 사진 자동 인식 | Google Vision API로 음식명 + 영양소 자동 추출 |
| AI | 영수증 OCR | Naver Clova OCR로 메뉴명 자동 추출 |
| AI | AI 메뉴 추천 | 오늘 영양 현황 기반 GPT 추천 + 부족 영양소 룰렛 가중치 |
| AI | 먹찌 일기 | GPT 자동 한 줄 코멘트 |
| 리텐션 | 스트릭 시스템 | 연속 기록 일수 표시 |
| 리텐션 | 주간 퀘스트 + 보상 | 업적 달성 조건 실시간 체크 및 보상 지급 |
| 리텐션 | 식사 패턴 리포트 | 주간/월간 식사 시간 통계 분석 |
| 리텐션 | 스마트 넛지 알림 | 시간·날씨·전날 식단 분석 기반 FCM 푸시 알림 |
| 리텐션 | 시즌 이벤트 | 계절 챌린지, 한정판 캐릭터 외형 |
| 소셜 | 친구 타임라인 피드 | 친구 식사 기록 공유 및 리액션 |
| 소셜 | 먹찌 카드 SNS 공유 | 내 먹찌 카드 이미지 생성 후 공유 |
| 캐릭터 | 외형 커스터마이징 | 획득 아이템으로 먹찌 꾸미기 |
| 캐릭터 | 뱃지 & 칭호 | 달성 조건별 칭호 부여 |
| 편의 | 지도/배달앱 딥링크 | 네이버 지도·배민 URL 스킴 연동 |
| 편의 | 홈화면 위젯 | 먹찌 배고픔 게이지 실시간 표시 |

### Tier 3 — 미래 확장

- 대화형 AI 영양 상담 (프리미엄)
- 그룹 챌린지 생성 및 현황 집계
- 그룹 메뉴 합의 (실시간 투표)
- 1:1 식단 배틀
- 위치 기반 로컬 퀘스트
- 바코드 스캔
- 음식 MBTI
- 식비 예산 관리
- Apple Watch / Wear OS 연동
