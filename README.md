# Mukzzi (뭐먹지)

> **"오늘 뭐 먹지?"** 를 해결해주는 AI 기반 음식 추천 & 캐릭터 성장 앱

---

## 서비스 소개

Mukzzi는 매일 반복되는 메뉴 고민을 해결하고, 먹은 음식으로 나만의 캐릭터 **먹찌(Mukzzi)** 를 성장시키는 게이미피케이션 식사 기록 앱입니다.
단순한 식단 관리를 넘어, 음식 선택이 곧 캐릭터 육성으로 이어지는 새로운 경험을 제공합니다.

---

## 핵심 플로우

### Step 1. 메뉴 추천 (The Choice)
- **AI 추천**: 현재 영양 현황을 분석해 부족한 영양소를 채워줄 메뉴를 추천
  - 예) *"오늘은 단백질이 부족해요. 캐릭터를 근육질로 키우고 싶다면 '스테이크'는 어때요?"*
- **랜덤 룰렛**: 결정 장애 해소 — 먹찌가 대신 룰렛을 돌려주는 인터랙티브 액션
- **이상형 월드컵**: 두 메뉴를 비교해 최종 승자를 선택하는 토너먼트 방식
- **상황별 필터**: 날씨, 기분(비 오는 날, 스트레스 등) 태그 기반 맞춤 추천

### Step 2. 식사 인증 (Feeding)
- **사진/영수증 인증**: 음식 사진을 업로드하면 AI 비전이 영양소를 자동 판별해 먹찌에게 경험치 부여
- **영수증 OCR**: 영수증 촬영으로 메뉴명 자동 추출
- **텍스트 직접 입력**: 메뉴명 검색으로 간편 등록

### Step 3. 캐릭터 성장 (Evolution)
- **실시간 반영**: 식사 등록 직후 경험치 바가 차오르고, 레벨 달성 시 진화 연출
- **영양소별 스탯 변화**: 섭취한 음식의 종류와 영양소에 따라 먹찌의 외형이 다르게 성장
- **패널티 시스템**: 장기 미기록 시 먹찌가 배고파하거나 기운이 없어지는 상태 노출

## 개발 로드맵

### [1단계] MVP — 핵심 기능 런칭

| 기능 | 설명 |
|------|------|
| 랜덤 룰렛 & 이상형 월드컵 | 메뉴 결정 시간을 0초로 단축하는 코어 기능 |
| 상황별 맞춤 필터 | 날씨·기분 태그 기반 추천 |
| 먹찌 즉각 변신 | 메뉴 입력 시 영양소 태그에 따른 캐릭터 변신 연출 (4~5종) |
| 먹찌 도감 | 변신했던 먹찌 외형 컬렉션 |
| 영양 밸런스 피드백 | 텍스트 기반 다음 메뉴 추천 멘트 |
| 먹부림 캘린더 | 날짜별 먹찌 아이콘 + 식사 기록 |
| 지도/배달앱 딥링크 | 네이버 지도·배민으로 URL 스킴 연동 |

### [2단계] Phase 2 — 리텐션 강화

| 기능 | 설명 |
|------|------|
| AI 음식 사진 자동 인식 | 비전 API를 활용한 사진 기반 자동 영양소 분석 |
| 스마트 넛지 알림 | 시간·날씨·전날 식단 분석 기반 푸시 알림 |
| 주간 퀘스트 시스템 | 먹찌 아이템 보상을 통한 재방문 유도 |
| 그룹 메뉴 합의 | 카카오톡 공유 + 실시간 투표 기능 |
| 냉장고 파먹기 | 보유 재료 기반 메뉴 추천 필터 |

### [3단계] Phase 3 — 생태계 & 수익화

| 기능 | 설명 |
|------|------|
| 대화형 AI 검색 | *"매운 거 1만 원 이하 찾아줘"* 자연어 검색 |
| 유저 메뉴 플레이리스트 | 리스트 생성·공유 소셜 커뮤니티 |
| 식비 예산 관리 | 영양 + 가성비 통합 가계부 |
| 할인 쿠폰 가챠 | 주변 식당 제휴·광고 수익 모델 |

## 주요 기능 상세

### AI 음식 인식 / 추천
- 음식 사진 분석 → 음식명 + 영양소 자동 추출
- 영수증 OCR → 메뉴명 추출
- 오늘 영양 현황 기반 AI 메뉴 추천
- 부족 영양소 메뉴 가중치 부스트 룰렛
- 대화형 AI 영양 상담 (프리미엄)

### 소셜
- 친구 식사 타임라인 피드 및 리액션
- 그룹 챌린지 생성 및 진행 현황 집계
- 1:1 식단 배틀 매칭 및 결과 처리
- 친구 초대 알림: *"OO가 음식을 기록했어요!"*

### 업적 / 퀘스트
- 업적 달성 조건 실시간 체크 및 보상 지급
- 위치 기반 로컬 퀘스트 생성 및 완료 처리
- 주간 목표 / 월간 도전 / 장기 목표 설정
- 시즌 이벤트 (계절 챌린지, 한정판 캐릭터 외형)

### 보상 & 성취 시스템
- 캐릭터 레벨업 → 새로운 의상 / 미니게임 / 필터 해금
- 뱃지 & 칭호 (*"근육질 완성"*, *"전설의 캐릭터"* 등)
- 시즌 이벤트 한정 보상 (기간 제한 미션)

### 시간 / 습관 추적
- 규칙적 식사 시간 알림 설정
- 식사 시간 통계 (평균 식사 시간대 분석)
- 3일 이상 미기록 시 먹찌 배고픔 알림
- 도전과제 알림: *"단백질 충분히 먹으셨어요. 내일은 채소는?"*

- 자주 검색되는 음식 캐싱 처리

## 기술 스택

### Mobile (Frontend)
| 항목 | 선택 | 이유 |
|------|------|------|
| 프레임워크 | React Native | iOS/Android 동시 개발, Kakao SDK 지원 |
| 상태관리 | Zustand | 가볍고 직관적 |
| 애니메이션 | Reanimated 3 | 먹찌 변신 연출에 적합 |

### Backend
| 항목 | 선택 | 이유 |
|------|------|------|
| 프레임워크 | Spring Boot (Kotlin) | JVM 기반, IntelliJ 환경, 한국 개발 생태계 |
| 실시간 | WebSocket (STOMP) | 그룹 챌린지, 배틀 실시간 처리 |
| 스케줄러 | Spring Scheduler | 패널티 시스템, 푸시 알림 배치 |

### Database
| 항목 | 선택 | 이유 |
|------|------|------|
| 메인 DB | PostgreSQL | 복잡한 영양소·캐릭터 관계 데이터 |
| 캐싱 | Redis | 자주 검색되는 음식 캐싱, 세션 관리 |
| 파일 저장 | AWS S3 | 음식 사진, 캐릭터 이미지 |

### AI / 외부 API
| 항목 | 선택 | 이유 |
|------|------|------|
| 음식 사진 인식 | Google Vision API | 정확도 높은 음식 분류 |
| 영수증 OCR | Naver Clova OCR | 한국 영수증 인식 최적화 |
| 메뉴 추천 AI | OpenAI GPT API | 대화형 영양 상담, 자연어 추천 |
| 음식 영양소 DB | 식약처 API + USDA | 국내외 음식 영양 정보 |

### 인증
| 항목 | 선택 |
|------|------|
| 토큰 | JWT (Access + Refresh) |
| 소셜 | Kakao OAuth 2.0 / Google OAuth / Apple Sign In |

### 알림 / 인프라
| 항목 | 선택 |
|------|------|
| 푸시 알림 | FCM (Firebase Cloud Messaging) |
